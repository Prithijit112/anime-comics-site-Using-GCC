Green Cloud Computing
Overview
This project demonstrates Green Cloud Computing by hosting a website on a solar-powered Raspberry Pi. 
The project highlights how renewable energy can be integrated into cloud infrastructure to reduce carbon footprints and operational costs.

Technologies
Raspberry Pi (Quad-core 600MHz, 512MB RAM)
HTML, CSS, JavaScript, Bootstrap (Frontend development)
Solar Power (Energy source)

Features
Solar-powered Web Hosting: A website hosted on a Raspberry Pi powered entirely by solar energy.
Bootstrap-based website: Lightweight, responsive, and energy-efficient with an anime comics theme.

SEO: 100
Accessibility: 72
